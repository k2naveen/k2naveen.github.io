# naveenk2.github.io

Personal blog — Linux graphics, Wayland, VRR.

## Local preview

```
sudo apt install ruby-full build-essential zlib1g-dev
bundle install
bundle exec jekyll serve
# open http://127.0.0.1:4000
```

## Publish

Push `main` to `https://github.com/naveenk2/naveenk2.github.io`.
GitHub Pages will build and serve automatically.
